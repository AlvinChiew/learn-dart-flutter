## Remarks

add Google Maps Platform `API KEY` to `com.google.android.geo.API_KEY` in file `AndroidManifest.xml`, e.g.

```xml
<meta-data android:name="com.google.android.geo.API_KEY"
    android:value="YOUR_API_KEY"/>
```
